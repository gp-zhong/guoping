# 截屏分析助手 安装指南

## 安装步骤

1. 下载并解压本扩展包
2. 打开 Microsoft Edge 浏览器
3. 在地址栏输入 edge://extensions/
4. 打开右上角的"开发人员模式"开关
5. 点击"加载解压缩的扩展"按钮
6. 选择解压后的扩展文件夹
7. 安装成功后，浏览器右上角会出现扩展图标

## 使用方法

1. 点击浏览器右上角的扩展图标打开侧边栏
2. 在设置中配置您的 DeepSeek API Key
3. 使用"立即截图"按钮手动截取网页
4. 或设置自动截图时间间隔，点击"开始"按钮自动截图
5. 点击截图缩略图可以查看详情和进行分析

## 快捷键

- Ctrl+Shift+S: 打开截屏分析助手侧边栏
- Ctrl+Shift+X: 立即截取当前页面

## 注意事项

- 如果扩展图标不可见，可以点击浏览器右上角的扩展菜单，找到"截屏分析助手"
- 使用图像分析功能需要有效的 DeepSeek API Key
- 截图数据保存在浏览器本地存储中
